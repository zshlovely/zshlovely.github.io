Hive SQL总结

1. 自带 function 的使用

使用show functions; 命令进行查看目前可以使用的function;

使用如下命令查看使用方法 split为例：

  • desc function extended split;

2. 常用func

  • LIKE

    A LIKE B

    如果字符串A或者字符串B为NULL，则返回NULL；如果字符串A符合表达式B的正则语法，则为TRUE；否则为FALSE。B中字符”_”表示任意单个字符，而字符”%”表示任意数量的字符。

  • RLIKE / REGEXP 二者等价

    A RLIKE B

    如果字符串A或者字符串B为NULL，则返回NULL；如果字符串A符合JAVA正则表达式B的正则语法，则为TRUE；否则为FALSE。

    例：select 1 from dual where ‘footbar’ rlike ‘^f.*r$’

  • +、-、*、/ 、% 加 减 乘 除 取余 操作

  • &、|、~、^ 与或非，异或操作

  • 复合类型

    map, struct, array

  • round(double a) 取整函数 四舍五入

  • round(double a, int d) 返回指定精度d的double类型

  • floor(double a) 向下取整函数

  • ceiling / ceil(double a) 向上取整函数

  • power / pow(double a, double p) 返回a的p次幂

  • from_unixtime(1323308943,’yyyyMMdd’) 时间戳转日期函数

  • unix_timestamp() 当前UNIX时间戳

  • unix_timestamp(’2011-12-07 13:01:03′) 日期转UNIX时间戳

  • unix_timestamp(’20111207 13:01:03′,’yyyyMMdd HH:mm:ss’) 指定格式日期转UNIX时间戳

  • to_date(’2011-12-08 10:03:01′)日期时间转 日期 out:2011-12-08

  • year(’2011-12-08 10:03:01′) 日期转年 可以直接做加减

  • month(’2011-08-08′)

  • day(’2011-12-24′)

  • hour(’2011-12-08 10:03:01′)

  • minute(’2011-12-08 10:03:01′)

  • second(’2011-12-08 10:03:01′)

  • weekofyear(’2011-12-08 10:03:01′)

  • datediff(’2012-12-08′,’2012-05-09′) 日期差

  • date_add(’2012-12-08′,10) 日期增加

  • date_sub(’2012-12-08′,10) 日期减少

  • if(1=2,100,200) 条件函数 out:200

  • COALESCE(null,’100′,’50′) 非空查找 out:100

  • 条件判断函数1：CASE

    语法: CASE a WHEN b THEN c [WHEN d THEN e]* [ELSE f] END

    如果a等于b，那么返回c；如果a等于d，那么返回e；否则返回f

    例：Select case 100 when 50 then ‘tom’ when 100 then ‘mary’ else ‘tim’ end from dual;

  • 条件判断函数2：CASE

    语法: CASE WHEN a THEN b [WHEN c THEN d]* [ELSE e] END

    如果a为TRUE,则返回b；如果c为TRUE，则返回d；否则返回e

    例如：select case when 1=2 then ‘tom’ when 2=2 then ‘mary’ else ‘tim’ end from dual;

  • length(string A) 字符串长度

  • reverse(string A) 字符串反转

  • 字符串连接函数：concat

    concat(string A, string B…)

  • 带分隔符字符串连接函数：concat_ws

    concat_ws(string SEP, string A, string B…)

    例如：concat_ws(‘,’,’abc’,’def’,’gh’)

  • 字符串截取函数：substr,substring

    substr(‘abcde’,3) out:abc

    substr(‘abcde’,-1) out:e

    substr(string A, int start, int len),substring(string A, int start, int len)

  • 字符串转小写函数：lower,lcase

    lower(string A) lcase(string A)

  • 去空格函数：trim

    trim(string A)

    左边去空格函数：ltrim

    右边去空格函数：rtrim

  • 正则表达式替换函数：regexp_replace

    regexp_replace(string A, string B, string C)

    将字符串A中的符合java正则表达式B的部分替换为C。注意，在有些情况下要使用转义字符

    regexp_replace(‘foobar’, ‘oo|ar’, ”) out:fb

  • 保留两位小数

    cast(column_name as decimal(10,2)) cast函数截取（推荐使用）

  • 列转行 （collect_all() / collect_list() 不去重）、（collect_set去重） 一般与concat_ws连用，带分隔符

    select col1, concat_ws(‘,’, collect_set(col2))
    from tb_name
    group by col1;

  • Hive窗口函数

    • row_number over（）

      row_number over(partition by member_code order by PT_BIZ_DATE,trade_no) num

    • sum() over ()

      sum(amount) over(partition by name order by mons rows between unbounded preceding and current row) as account

      对name这一组按照月份排序

      rows between unbounded preceding and current 限定了行是按照在当前行不限定的往前处理, 通俗就是处理当前以及之前的所有行的sum

      unbounded意思无限的 preceding在之前的，current row当前行。

    • count() over ()

      select count(*) over(partition by col1,col2) from tmp_test;

4.hive日期整理

  1. 日期函数

    获取当前日期：
    SELECT CURRENT_DATE
    2018-03-22

    select datediff(CURRENT_DATE,’2017-01-01’) as datediff;
    select date_add(current_date,365) as dateadd;

    add_months(CURRENT_DATE,-12) 去年今天
    last_day(current_date) 本月最后一天

    yyyymmdd和yyyy-mm-dd日期之间的切换————————————
    方法1: from_unixtime+ unix_timestamp
    –20171205转成2017-12-05
    select from_unixtime(unix_timestamp(‘20171205’,’yyyymmdd’),’yyyy-mm-dd’) from dual;

    –2017-12-05转成20171205
    select from_unixtime(unix_timestamp(‘2017-12-05’,’yyyy-mm-dd’),’yyyymmdd’) from dual;

    方法2: substr + concat
    –20171205转成2017-12-05
    select concat(substr(‘20171205’,1,4),’-‘,substr(‘20171205’,5,2),’-‘,substr(‘20171205’,7,2)) from dual;

    –2017-12-05转成20171205
    select concat(substr(‘2017-12-05’,1,4),substr(‘2017-12-05’,6,2),substr(‘2017-12-05’,9,2)) from dual;

rank() 排序相同时会重复，总数不会变

dense_rank() 排序相同时会重复，总数会减少

row_number() 会根据顺序计算

select member_code, trade_time from
(select member_code, trade_time,
row_number() over ( partition by member_code order by trade_time,trade_no) num
from APP.cms_orders_lrp where source=1 and trade_amount>0 and order_type!=’Redemption Order’
) temp
where temp.num = 1

玩家id,游戏包名，交易完成时间,购买礼包id,交易币种,成交金额,交易日期

求5月每个玩家最早的三次付费item和金额

SET @num=1;
SET @last_id=0;
SELECT id, 1_item,1_usd,IFNULL(2_item,null) 2_item,IFNULL(2_usd,0) 2_usd,IFNULL(3_item,null) 3_item,IFNULL(3_usd,0) 3_usd
FROM (
select
id,
GROUP_CONCAT(if(seq=1,item,NULL)) AS 1_item,
GROUP_CONCAT(if(seq=1,usd,NULL)) AS 1_usd,
GROUP_CONCAT(if(seq=2,item,NULL)) AS 2_item,
GROUP_CONCAT(if(seq=2,usd,NULL)) AS 2_usd,
GROUP_CONCAT(if(seq=3,item,NULL)) AS 3_item,
GROUP_CONCAT(if(seq=3,usd,NULL)) AS 3_usd
from(
SELECT t1.*,IF(@last_id=id,@num:=@num+1,@num:=1) AS seq,
(@last_id:=id) AS tmp FROM (
SELECT id,time,item,usd
FROM purchase
where month(time)=5
GROUP BY id ASC,time ASC
) t1
)t2 GROUP BY id
) t3
